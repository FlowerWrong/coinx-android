# coinx-android
该项目目前已经结合了okex,bitmex平台的合约行情，合约交易等API的实现。 项目获取的平台APIKEY和APISECRET都存放在手机本地，绝不会上传到其他地方，不放心可以直接看源码。

如果使用过程中发现什么问题可以反馈给我们，我们会第一时间协助处理。

# 功能
1. 包行两个平台各自的币种排行 
2. 可对币种进行添加自选操作，方便查找 
3. 可查看币种的k线和交易历史 
4. 可对合约币种进行开平仓操作  
5. 可查看委托历史和持仓记录 
6. 可查看绑定账号的账户信息 具体可下载下来体验。
